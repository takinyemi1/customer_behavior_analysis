# customer_behavior_analysis
This project analyzes customer shopping behavior using transactional data from 3,900 purchases across various product categories. The goal of this project is to uncover insights into spending patterns, customer segments, product preferences, and subscription behavior to guide strategic business decisions using Python, SQL, and Power BI.

# DATASET
- [x] Source: https://github.com/takinyemi1/customer_behavior_analysis/blob/main/customer_shopping_behavior.csv
- [x] Size: 3,900 rows | 18 columns
- [x] Description: There are customer demographics such as age, gender, location, and subscription status. Purchase details consist of items purchased, category, purchase amount, season, size, and color. Shopping behavior details include  discount(s) applied, promo code(s) used, previous purchases, purchase frequency, review ratings, and shipping type used for online purchases. Our missing data consists of 37 values from the ‘Review Rating’ column.
